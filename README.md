# Mobile_Sales_Dashboard-
PowerBi Dashboard 
This project delivers dynamic business insights using interactive KPIs, monthly transaction analysis, quantity trends by month, and quarter-wise comparison with previous year data. The dashboard is built to update dynamically as new data is added, ensuring real-time performance tracking.

Power Query is used for data cleaning, transformation, and integration from multiple sources including Excel and SQL. A robust data model is implemented to support accurate reporting, trend analysis, and performance comparison.

Key Features:

Dynamic KPIs for real-time performance monitoring

Monthly transaction and quantity analysis

Quarter-wise comparison with previous year data

Automated data refresh and dynamic visuals

Data cleaning and transformation using Power Query

Data integration from Excel and SQL

Optimized data modeling for accurate insights
